不用tui image editor了，自己手搓一个图片编辑器，参考snipaste的交互
- [ ] 图片编辑功能

  - [ ] 图片裁剪优化

    - [ ] 点击裁剪，用户划选支持显示裁剪范围，支持调整，按enter进行裁剪
    - [ ] 裁剪后再次点击裁剪，需要显示原图大小和裁剪范围，方便调整裁剪范围
    - [ ] 裁剪的图片保存后，再编辑，可以重新调整裁剪范围(注意需要存储原图的尺寸，这样调整裁剪范围可以正确显示原图，而不是以裁剪范围的大小显示原图，导致原图被压缩）
  - [ ] 图片水平和垂直翻转
  - [ ] 图片旋转

    - [ ] 默认90°旋转，无法修改
    - [ ] 悬浮工具栏支持设置

      - [ ] 顺时针旋转
      - [ ] 逆时针旋转
  - [ ] 添加形状工具：矩形、圆形

    - [ ] 点击形状工具，默认选中矩形工具，这样可以快速添加矩形
    - [ ] 悬浮工具栏支持设置

      - [ ] 改成矩形还是圆形
      - [ ] 粗细
      - [ ] 颜色
    - [ ] 激活形状工具情况下，点击已经添加的矩形和圆形，支持直接修改粗细、颜色，还能移动形状，但不能修改其他工具添加的东西，其他工具也是类似
  - [ ] 添加序号工具（圆形+数字样式）

    悬浮工具栏支持设置

    - [ ] 支持设置背景色，默认背景色是红色， 字体为白色
    - [ ] 支持设置序号编号，每次添加自动递增序号编号
    - [ ] 用户点击序号，支持修改背景色和编号
  - [ ] 添加直线和箭头工具

    悬浮工具栏可以设置

    - [ ] 两端是否开启箭头，直接一个下拉框，有以下几个选项

      - [ ] 无镜头
      - [ ] 左边箭头⬅️
      - [ ] 右边箭头→
      - [ ] 两边都有箭头 ↔️
    - [ ] 支持调整粗细和颜色
  - [ ] 添加马赛克

    悬浮工具栏

    - [ ] 设置马赛克程度
  - [ ] 添加画笔、橡皮擦

    悬浮工具栏

    - [ ] 设置颜色和粗细
  - [ ]  添加文字工具
    - [ ]  支持设置字体
    - [ ]  支持设置大小
    - [ ]  支持设置颜色

  - [ ] 支持鼠标选择
  - [ ] 支持抓手功能，按空格加鼠标左键支持直接移动画布（不更改当前工具）

  - [ ] 支持撤回和恢复、重置功能
